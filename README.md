# webpack4-typescript-template

#### Description

基于 Vue-cli@2 脚手架升级改造的全 typescript 实现

#### 技术栈

JS、TS、TSX、Vue 全家桶

#### 开发支持

Eslint、Babel、PostCss

#### 关键第三方依赖版本

```json
"typescript": "^3.7.4",
"ts-node": "^8.5.4",
"vue": "^2.6.11",
"webpack": "^4.41.2",
"@babel/core": "^7.7.5",
"eslint": "^6.7.2",
```

#### 额外支持

markdown、json5

#### 安装教程

1.  npm install
2.  npm run dev
3.  npm run build

#### build for production and view the bundle analyzer report

npm run analyzer
